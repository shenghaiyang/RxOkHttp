# RxOkHttp

RxJava and OkHttp

## Usage

- gradle

```
compile 'com.shenghaiyang.rxokhttp:rxokhttp:1.0.0'
```

```
compile 'com.shenghaiyang.rxokhttp:rxokhttp-gson:1.0.0'
```
